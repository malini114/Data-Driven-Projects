# Single Women Demographic Analysis - Tamil Nadu

## Project Overview
This project analyzes census data to understand the demographic profile of single women in Tamil Nadu.  
Key variables include age, education, employment, area type and marital status.

## Objectives
- Clean and transform census data using **Excel + Power Query**
- Apply **DAX expressions** for calculated measures
- Build interactive dashboards in **Power BI**

## Tools & Skills
- **Excel** → Data cleaning and preparation  
- **Power Query** → ETL workflows  
- **DAX Expressions** → Custom measures and calculations  
- **Power BI** → Interactive dashboards and visualizations  

## Key Insights
The Power BI dashboard highlights the following:
- **Overall population, total women population, and total single women population**  
- **Single women by age group** distribution  
- **District-wise analysis** → lowest and highest single women population  
- **Single women by education and employment status**  
- All insights are **filterable by census year (2001, 2011)** and **area type (Rural, Urban, Total)**
